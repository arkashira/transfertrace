# TransferTrace

TransferTrace is a CLI tool for monitoring file transfer progress with a real-time progress bar. It provides detailed information about the transfer status, including percentage completed, transferred bytes, total size, and estimated time remaining.

## Features

- Real-time progress bar in the CLI
- Displays percentage completed, transferred bytes, total size, and estimated time remaining
- Pauses and resumes transfer progress tracking
- Non-interfering with existing logging output
